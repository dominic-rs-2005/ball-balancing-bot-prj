**Ball Balancing Robot**

This repository contains the complete implementation of a Ball Balancing Robot, a control systems project focused on real-time stabilization using feedback control.

The system balances a ball on a platform by dynamically adjusting its orientation using a PID (Proportional–Integral–Derivative) controller. A resistive touch sensor detects the ball’s position, and an Arduino processes this data to calculate error and generate control signals. Servo motors then tilt the platform to bring the ball back to the desired position, forming a closed-loop control system.

* Closed-loop PID control for accurate stabilization
* Servo motor-based platform actuation
* Smooth and stable performance with minimal oscillations
